Interactive Content top &aurr;

Conteúdo que permite a interação do usuario Nota: Alguns elementos no HTML podem ser ativados por um comportamento. Isso significa que o usuario pode ativa-lo de alguma forma. O inicio da sequencia de eventos depende do mecanismo de ativação e normalmente culminam em um evento de click seguido pelo evento DOMAActivate. O user-agent permite que o usuario ative manualmente o elemento que tem este comportamento utilizando um teclado, mouse, comando de voz, etc.

  • audio (se o atributo Control
  • canvas
  • embed
  • iframe
  • img (se o atributo usemap for ultilizado
  • input (se o atributo type não tiver o valor hidden
  • keygen
  • label
  • menu (se o atributo type tiver o valor toolbar
  • object (se o atributo usemap for ultilizado
  • select
  • textarea
  • video (Se o atributo control for ultilizado
